# 龙心 OS (Dragon Heart OS)

> **AI智能体调度中枢** — 1+5 模式，感知上下文→识别场景→自动调度引擎

## 概述

龙心操作系统（Dragon Heart OS）是 AI 智能体的**发动机与调度中枢**，采用 **1+5 模式**：

- **1 个总智能体**：龙心 OS 调度中枢
- **5 个子智能体引擎**：
  - 知行合一 —— 概念落地转化
  - 知识学习 —— 深度认知学习
  - 人机协同五象限 —— 人机协作模式 + 执行层标注
  - 象思维 —— 基于易经的认知框架
  - 五色光思维 —— 结构化决策管理

## 核心特性

- **真正的自动触发**：感知上下文→识别场景→自动调度引擎
- **人机协同五象限 v4.0**：三层架构（战略→执行→优化）
- **AI原生组织工作流**：任务级[AI]/[人]/[⇄]三分类标注
- **本能三问**：每次任务自动判断，无需手动加载
- **场景分类器**：自动识别 15+ 类场景
- **引擎路由**：智能分发到对应子智能体
- **自主进化闭环**：EVOLUTION.md + GAPS.md 持续迭代
