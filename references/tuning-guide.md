# 龙心OS调优指南

## 场景识别参数调优

### 置信度阈值
| 场景 | 阈值 | 说明 |
|------|:----:|------|
| S0日常 | 60% | 低阈值，快速响应 |
| S1任务 | 80% | 确保任务理解准确 |
| S2深度 | 70% | 知识学习场景 |
| S3创意 | 75% | 象思维场景 |
| S4决策 | 85% | 需高准确率 |
| S5重大 | 90% | 最高阈值 |

### 引擎调度权重
- 人机协同五象限: 基础层权重30%
- 知识学习: 深度理解层权重25%
- 象思维: 创意创新层权重20%
- 知行合一: 落地执行层权重15%
- 五色光思维: 决策分析层权重10%

## 反馈闭环
每次调度后记录准确率，自动调整权重。
